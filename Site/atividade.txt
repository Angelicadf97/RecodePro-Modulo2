Olá, Estudante!  
Você começa agora o seu mais novo projeto, a ser realizado individualmente!
Uma agência de viagens precisa de você para desenvolver uma série de módulos para organizar
as informações dos seus clientes, apresentar seus serviços e melhorar os serviços oferecidos
aos seus parceiros. 
Vamos começar? 
Nesta atividade, vamos usar os conceitos de algoritmos em pseudocódigo (Resolução de problemas
usando portugol), HTML, CSS e BOOTSTRAP para a construção de um modelo de agência de viagens.             

Usando HTML / CSS / BOOTSTRAP: 
Criar um protótipo da primeira versão do site; 
Criar as páginas: Home, Destino, Promoções, Contato e as demais páginas de acordo com seu
modelo de negócio. Usando HTML, CSS e BOOTSTRAP; 
As páginas devem ser responsivas ou adaptáveis. 
Faça as formatações de acordo com a paleta de cores escolhida.
